# 📚 Note Struktur Data

Website sederhana berbasis HTML dan CSS yang berisi dokumentasi materi dan catatan pembelajaran Struktur Data selama semester 2 Teknik Informatika.  
Project ini dibuat sebagai media belajar pribadi agar materi lebih terstruktur, mudah dibaca, dan lebih nyaman dipelajari dibanding catatan biasa.

---

## ✨ Tentang Project

Berisi dokumen tasi materi dan catatan pembelajaran Struktur Data selama semester 2 Teknik Informatika, disusun sebagai media belajar yang kurang ringkas, tidak terstruktur, dan sulit dipahami.

Website ini fokus pada:
- Dokumentasi materi
- Catatan pembelajaran
- Arsip modul semester 2
- Media belajar mandiri

---

## 🛠️ Tech Stack

Project ini dibuat menggunakan:

- HTML5
- CSS3

Tanpa framework tambahan seperti Bootstrap atau Tailwind.
