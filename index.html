Menuju Materi

Learning Modules

Stack, Queue, Deque

Membahas Stack (LIFO), Queue (FIFO), dan Deque yang dapat mengakses data dari dua sisi.

Sorting #1

Membahas Bubble Sort, Selection Sort, dan Insertion Sort sebagai algoritma dasar pengurutan data.

Rekursif

Membahas fungsi rekursif yang memanggil dirinya sendiri untuk menyelesaikan masalah secara bertahap.

Searching

Membahas teknik searching untuk mencari data menggunakan Sequential Search dan Binary Search.

Hashing

Membahas hashing untuk menyimpan dan mencari data dengan cepat menggunakan fungsi hash.

Sorting #2

Membahas algoritma sorting seperti Merge Sort, Quick Sort, dan Counting Sort untuk mengurutkan data secara efisien.
